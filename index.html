連携および各種設定

Google API 連携設定

GoogleカレンダーおよびGmailと接続するためのAPI情報を設定します。情報はご自身のブラウザ内にのみ安全に保存されます。

Claude AI 設定

AIスケジュール提案・メール解析機能に使用する Claude API キーを設定します。 キーは console.anthropic.com から取得できます。 情報はブラウザ内にのみ保存されます。

※ APIキーはブラウザのローカルストレージにのみ保存され、Claude API呼び出し時にのみ使用されます

メールから予定を自動取得

Gmailの未読メールをAIが自動解析し、日程・会議・イベントなどの予定を検出してカレンダーに追加します。 Google連携とClaude APIキーの両方が必要です。

このアプリの使い方

1

予定とタスクの使い分け

「予定」は会議や面談など特定の時間枠を占有するもの。「タスク」は重要度や期日に基づいて優先度が決まるやるべきことです。

2

AIスケジュール提案

「本日の予定」タブの「AIスケジュール提案」ボタンを押すと、登録済みのタスクと予定をAIが分析し、最適な作業時間帯を提案します。

3

メールから予定を自動取得

設定画面の「メールから予定を取得」ボタンで、Gmailの未読メールをAIが解析し、含まれる日程をカレンダーに自動追加できます。

4

Gmail連携と返信処理

フィードバックが必要なタスクには「返信する」ボタンが表示され、件名と宛先が入ったGmail送信画面が直接開きます。
